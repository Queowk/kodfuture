## 📝 Описание программы

Weather Diary - это консольное приложение для ведения дневника наблюдений за погодой. Программа позволяет:

- ✅ Добавлять, просматривать и удалять записи о погоде
- 🔍 Фильтровать записи по дате или температуре
- 📈 Строить графики температуры и осадков с помощью библиотеки matplotlib
- 💡 Получать полезные советы на основе погодных условий
- 💾 Сохранять и загружать данные в формате JSON
- 📊 Просматривать статистику (средняя, максимальная, минимальная температура)

Приложение построено с использованием принципов объектно-ориентированного программирования (инкапсуляция, наследование, полиморфизм) и включает обработку ошибок ввода.
